# AddictionHelp

AddictionHelp is a mobile-first streak tracker built around Ironhorn, a bison guardian who grows stronger as users continue their recovery streak.

## First version

- Home screen with the current streak
- Ironhorn character card with placeholder art
- Level progression based on streak days
- Daily completion button
- Supportive restart flow that preserves lifetime XP

## Development

```bash
npm install
npm run dev
```

Open `http://localhost:3000` to view the app.
